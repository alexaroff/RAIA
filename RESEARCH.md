# Исследования проекта RAIA

## О проекте

RAIA (Recruitment AI Assistant) — локальный помощник рекрутера для автоматизации обработки интервью и анализа кандидатов.

Текущее оборудование:

* MacBook Air M2
* 8 ГБ ОЗУ

Стек:

* Python
* WhisperX
* Ollama
* FFmpeg

---

## Цель

Получить локальную транскрипцию интервью на русском языке.

## Результат

Успешно настроены:

* Python Virtual Environment
* WhisperX

# Исследование №2 — WhisperX Turbo

## Задача

Улучшить качество транскрипции по сравнению со стандартной моделью.

---

### Проблема №1

Ошибка загрузки модели через Hugging Face.

Ошибка:

ReadTimeoutError

### Решение

Модель скачана вручную через curl.

Локальное расположение:

~/whisper_models/turbo/

---

### Проблема №2

Конфликт версий:

* transformers
* huggingface_hub

Ошибка:

ImportError

### Решение

Выполнен откат:

pip install huggingface-hub==0.34.0

---

### Проблема №3

Конфликт:

* PyTorch 2.8
* Pyannote

Ошибка:

_pickle.UnpicklingError

### Решение

Отказ от Pyannote.

Переход на:

--vad_method silero

---

### Проблема №4

Высокая нагрузка на MacBook Air M2.

### Оптимизация

Увеличены параметры:

* batch_size
* chunk_size

Ограничены потоки CPU:

export OMP_NUM_THREADS=2

Добавлена очистка аудио через FFmpeg.

# Главный технический вызов

На текущем этапе проблема заключается не в качестве транскрипции, а в определении ролей участников разговора.

Необходимо автоматически разделять:

* Рекрутер
* Кандидат
* Нанимающий руководитель

без ручной разметки интервью.
