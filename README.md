
# RAIA — Recruitment AI Assistant
 
## О проекте
 
RAIA (Recruitment AI Assistant) — исследовательский HR Tech проект по автоматизации процессов рекрутмента с использованием технологий искусственного интеллекта.
 
Проект создаётся как персональная площадка для изучения возможностей AI в подборе персонала и построения собственного HR-ассистента.
 
> Искусственный интеллект помогает рекрутеру принимать решения, но не заменяет человека.
 
---
 
## Какие задачи решает проект
 
В процессе подбора персонала значительное время уходит на:
 
- расшифровку интервью;
- разметку ролей участников (рекрутер / кандидат / руководитель);
- написание саммари и анализ компетенций;
- подготовку вопросов на следующий этап;
- формирование HR-отчётов.
Цель RAIA — автоматизировать рутинные задачи и сократить время обработки интервью от записи до готового отчёта.
 
---
 
## Текущий рабочий процесс
 
Проект развивается поэтапно. Транскрипция реализована локально, HR-анализ пока выполняется вручную через облачный DeepSeek.
 
```
Аудио интервью
      ↓
[Локально] WhisperX Turbo + Silero VAD
      ↓
Транскрипт (TXT / JSON / SRT)
      ↓
[Вручную / DeepSeek] Разметка ролей: HR / CAND
      ↓
Структурированный диалог
      ↓
[DeepSeek] HR-анализ по промту: Вакансия + CV + Транскрипт
      ↓
HR-отчёт → Решение рекрутера
```
 
### Что автоматизировано
 
- Локальная транскрипция через WhisperX Turbo
- Поддержка русского языка
- Экспорт в TXT, JSON, SRT
- Оптимизация под MacBook Air M2 (8 ГБ ОЗУ)
### Что пока делается вручную
 
- Разметка ролей участников интервью
- Передача транскрипта в DeepSeek
- Генерация HR-отчёта по промту

- DeepSeek API — разметка ролей и HR-анализ

- Gemma 4 12B
