# RAIA — Recruitment AI Assistant / Ассистент "Райя"

## О проекте

RAIA (Recruitment AI Assistant) — исследовательский HR Tech проект по автоматизации процессов рекрутмента с использованием технологий искусственного интеллекта.

Проект создаётся как персональная площадка для изучения возможностей AI в подборе персонала и построения собственного HR-ассистента.

> Искусственный интеллект помогает автоматизировать рутину рекрутеру и принимать решения, но не заменяет человека полностью.

---

## Какие задачи решает проект

В процессе подбора персонала значительное время уходит на:

- поиск и первичный скрининг резюме;
- проведение первого телефонного звонка;
- согласование времени интервью с кандидатом и командой;
- подготовку вопросов на интервью;
- проведение самого интервью;
- конспектирование скрининга/интервью;
- детальное конспектирование развёрнутых ответов и цитат кандидата;
- написание саммари и анализ компетенций/рисков;
- подготовку вопросов на следующий этап;
- формирование HR-отчёта для нанимающего менеджера;
- написание персонализированной обратной связи кандидатам;
- подготовку и проверку тестовых заданий/кейсов;
- сбор и верификацию рекомендаций;
- формирование и внутреннее согласование оффера;
- внесение статусов и комментариев в ATS/CRM;
- регулярные синхронизации с нанимающим менеджером.

Цель RAIA — автоматизировать и сократить время рутинных задач.

---

## Планируемый сценарий работы AI + Рекрутер

```
Загрузка аудио/видео интервью
      ↓
Транскрипт (TXT / JSON / SRT) 
      ↓
Разметка ролей: HR / CAND / Hiring Manager
      ↓
Структурированный диалог
      ↓
HR-анализ: Вакансия + CV + Структурированный диалог
      ↓
HR-отчёт → Решение (берём / условно / не берём)
      ↓
Персонализированная обратная связь кандидату
      ↓
Подготовка оффера / отказа
      ↓
Внесение статуса и комментариев в ATS/CRM
```
