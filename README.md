# RAIA — Recruitment AI Assistant

## О проекте

RAIA (Recruitment AI Assistant) — исследовательский HR Tech проект по автоматизации процессов рекрутмента с использованием технологий искусственного интеллекта.

Проект создаётся как персональная площадка для изучения возможностей AI в подборе персонала и построения собственного HR-ассистента.

Основной принцип проекта:

> Искусственный интеллект помогает рекрутеру принимать решения, но не заменяет человека.

---

## Какие задачи решает проект

В процессе подбора персонала значительное время уходит на:

* расшифровку интервью;
* подготовку рекомендаций по кандидатам;
* написание саммари;
* анализ компетенций;
* подготовку вопросов на следующий этап;
* формирование отчётов по интервью.

Цель RAIA — автоматизировать рутинные задачи и сократить время обработки интервью.

---

## Текущий статус

Стадия проекта:

**Research & Prototype**

Уже реализовано:

* локальная транскрипция интервью через WhisperX;
* поддержка русского языка;
* экспорт результатов в TXT, JSON и SRT;
* исследование локальных LLM через Ollama;
* оптимизация работы на MacBook Air M2 (8 ГБ ОЗУ).
